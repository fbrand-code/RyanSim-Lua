RyanSim is a short visual novel game set in the epilogue of VIOLA_ with a charming artstyle using Sora's Pixel Converter (https://pixel-converter.ameniwa.com/). 

It is built in Lua using LOVE2D as well as uses the Talkies framework.

To play the game, unzip the folder and select RyanSim.exe.

Built entirely in two days.
